Spot Me is a web app I designed that creates and saves Spotify playlists based on a user-specified 'seed' song. Once they select the song, they can manipulate energy, loudness, and danceability values to their liking. An additional feature allows users to follow a random playlist that corresponds to the time and date.

MATLAB Baseball uses the MATLAB programming language to simulate a full length baseball game based on statistics from the 2015 MLB season. Complete with some barebones graphics, it uses random numbers to generate probabalistically accurate occurrences.

Rezzy is a restaurant reservation template, similar to OpenTable. As it is a template, both the front and back end are currently accessible. Users can enter their desired reservation date,time, and party size, and will receive a confirmation email if their reservation can be accomodated. Rezzy uses a SQL database to store and retrieve data along with a PHP backend infrastructure. Other features include deleting reservatons and a "Current Reservations" section that shows currently available tables and the names and sizes of the parties that are currently eating.
